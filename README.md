# python-adventure

This repository documents my journey completing "100 Days of Code™: The Complete Python Pro Bootcamp" (Udemy). I will work through the course during 2026 and publish daily progress, projects, and notes here on GitHub.

**Project Goal**

- Complete the full course curriculum and build the projects in each module.
- Share daily progress, short write-ups, and code for each completed day.
- Use this repo as a public learning log and portfolio of Python projects.

**How I'll Track Progress**

- Daily entries: `days/day-XX/README.md` for each day (summary, notes, code).
- Scripts & code: `days/day-XX/scripts/` for runnable examples and exercises.
- Commits: one commit per day with clear messages like "Day 01: Blackjack GUI".
- Projects: larger multi-day projects live under `projects/` with README and usage notes.

See [days/day-00/README.md](days/day-00/README.md) as a template for daily entries — just copy the folder structure and update the content.

**Roadmap & Schedule**

- Follow the course structure: fundamentals, OOP, file I/O, web scraping, APIs, GUIs, web basics, and final projects.
- Aim to post progress daily. If I need catch-up days, I will note them in the daily logs.

This is a personal learning repository. Contributions, pull requests, and issues are not accepted — this repo is maintained for my private course progress and portfolio only.
