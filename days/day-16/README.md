# Day 16 — OOP

- Date: 2026-06-26
- Duration: 2 hr

## Summary

Day 16 introduces Object-Oriented Programming (OOP) in Python. The focus is on creating classes, instantiating objects, and using methods and attributes to model real-world behavior. The main exercise is building a coffee machine application using multiple classes and encapsulated behaviors.

## Repository Structure

The main files and folders for Day 16 are:
- `oop-coffee-machine-start/` — starter code and the main coffee machine challenge.
- `challenges/` — smaller OOP practice scripts and exercises.
- `main.py` — basic lesson file demonstrating OOP concepts.
- `NOTES.md` — Day 16 OOP notes and definitions.
- `Coffee+Machine+Classes+Documentation.pdf` — class design reference.
- `Coffee+Machine+Program+Requirements.pdf` — program requirements and expected behavior.

## What to practice

- Defining classes with `class`.
- Using `__init__` to initialize object state.
- Creating object instances from classes.
- Defining and calling methods.
- Accessing attributes on objects.
- Organizing code across multiple classes for cleaner design.
